About this Gallery

This interactive gallery showcases a collection of generative motion sketches built using p5.js. Developed while studying at University of the Arts London (UAL), these pieces explore natural movement, playfulness, and screen-based expression. From soft rain to digital leaves, each sketch offers a glimpse into how simple code can echo the gentle complexity of nature.
